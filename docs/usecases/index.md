## All Use Cases

| ID | Use Case | Actor | Description |
|---|---|---|---|
| UC-001 | Register Account | All Users | Create an account and select a role |
| UC-002 | Login to System | All Users | Authenticate and access role-based dashboard |
| UC-003 | Post Job Vacancy | Company | Create and submit a job posting for admin review |
| UC-004 | Browse/Apply for Jobs | Student | Search and apply for available positions |
| UC-005 | Create Screening Quiz | Company | Create a quiz attached to a job posting |
| UC-006 | Take Screening Quiz | Student | Complete a quiz as part of the application process |
| UC-007 | Assign to Interview | System | Auto-assign passing students to interview stage |
| UC-008 | Create Mock Exam | Company | Create a mock exam linked to a vacancy after CV collection |
| UC-009 | Take Mock Exam | Student | Take the timed online mock examination |
| UC-010 | Auto-Shortlist Candidates | System | Rank by score and shortlist top performers or those above cutoff |
| UC-011 | View Exam Analytics | Company | Review score distribution, pass rates, candidate performance |
| UC-012 | Post Freelance Project | Company | Post a project with milestones and budget |
| UC-013 | Accept Freelance Work | Student | Accept project and sign digital agreement |
| UC-014 | Process Payment | System | Release payment upon milestone approval |
| UC-015 | Book Appointment | Student | Book a session with a consultant |
| UC-016 | Manage Availability | Consultant | Set and manage schedule |
| UC-017 | Rate Consultant | Student | Provide rating and feedback after a session |
| UC-018 | Verify Company | Admin | Review and approve company registration |
| UC-019 | Submit Complaint | All Users | Submit and track a complaint or issue |
| UC-020 | Create CV/Resume | Student | Build a professional CV using templates and export as PDF |
| UC-021 | Attach CV to Application | Student | Select a saved CV when applying for a job vacancy |
| UC-022 | Create Fund Collection Campaign | Student | Create an event campaign with payment details and item options |
| UC-023 | Contribute to Campaign | Student | Make a payment and select item preferences |
| UC-024 | Manage Campaign | Student | Track contributions, export reports, manage refunds |
| UC-025 | Approve Event Campaign | Admin | Review and approve fund collection campaigns |
| UC-026 | Create and Publish Notice | Admin | Compose, configure targeting/priority, and publish or schedule |
| UC-027 | View Notices | All Users | View active notices, filter by category, mark as read |
| UC-028 | Schedule Notice | Admin | Set future publish date/time; system auto-publishes |
| UC-029 | Track Notice Read Receipts | Admin | View read receipt log to monitor engagement |
| UC-030 | Archive/Delete Notice | Admin | Unpublish, archive, or soft-delete; all actions logged |
| UC-031 | Request Password Reset | Company / Consultant | Submit password reset request to Campus Administrator |
| UC-032 | Admin Reset Password | Admin | Reset password to default; user forced to change on login |
