# Introduction

## Purpose

This documentation provides a comprehensive reference for the **NextStep Job Portal Management System**. It covers the software requirements specification (SRS), RESTful API endpoints, system architecture, data models, and use case definitions.

NextStep is intended for developers, testers, project managers, and all stakeholders involved in the development and deployment of the system.

**Project Repository**: [https://github.com/Imesh-Bandar/NextStepDoc.git](https://github.com/Imesh-Bandar/NextStepDoc.git)
:::

## What is NextStep?

NextStep is a centralized digital platform designed to connect **students**, **companies**, **career consultants**, and **campus administrators**. The system facilitates job discovery, freelance hiring, career guidance, and issue resolution while ensuring transparency, security, and accountability across all stakeholders.

The platform is focused on student employment, freelance project opportunities, and career development support within an academic or early-career ecosystem.

## Core Capabilities

| Capability | Description |
| ------------ | ---------------------------------------------------------------------------- |
| Job Vacancy Management | Internship and job management with quiz-based screening |
| Freelance Hiring | Secure milestone-based payment processing |
| Career Guidance | Appointment scheduling with consultant rating system |
| Examination System | Interview assignment for qualifying students |
| Mock Examinations | Candidate shortlisting with automated ranking and merit-based filtering |
| CV/Resume Builder | Professional templates with PDF export |
| Event Fund Collection | Campaign-based collection replacing traditional Google Forms workflows |
| Complaint Tracking | Issue tracking for all platform stakeholders |
| Notice Management | Targeted platform-wide or role-specific announcements |
